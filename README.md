<h2>Etch-A-Sketch</h2>


Utilizing the Turtle module, created a Etch-A-Sketch application. The controls to move the turtle are:
</br></br>
Up: "w"
</br>
Down: "s"
</br>
Turn left: "a"
</br>
Turn right: "d"
</br>
</br>
Reset Screen: "c"
</br></br>
